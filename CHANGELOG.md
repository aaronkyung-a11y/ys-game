## v0.22 (2026-07-15) — 주인공 행동 자연스러움 개선

**이동에 관성 추가**
- 기존: `x += ix*spd` — 속도가 0에서 최대로 즉시 점프. 출발/정지가 이진적.
- 변경: `vx/vy` 속도 상태 추가. 목표 속도로 가속(`ACCEL 0.28`), 입력을 놓으면 감속(`DECEL 0.22`).
- 결과: 출발 시 3.7 → 5.0 램프업, 정지 시 2.4 → 0 활강.

**방향 전환 보간**
- 기존: 입력 즉시 facing 벡터가 스냅 — 로봇처럼 꺾임.
- 변경: `fang`(각도) 상태 추가, 최단 회전 방향으로 프레임당 `TURN_RATE 0.35` 보간.
- 결과: 0 → -1.57 전환이 4프레임에 걸쳐 부드럽게.

**공격 중 전진 (lunge)**
- 기존: 스윙 중 완전 정지 (`atkT>0`이면 이동 코드 자체를 건너뜀).
- 변경: 액티브 프레임 동안 facing 방향으로 전진력 추가 (일반 1.5, 대시공격 3.2). 스윙 중 속도는 0.80씩 감쇠.
- 결과: 칼을 휘두르며 몸이 따라 들어감. 이스 특유의 "몸통 박치기" 느낌에 근접.

**대시 감속 곡선**
- 기존: `DASH_SPD` 고정 속도로 11프레임 슬라이드 — 얼음판 위를 미끄러지는 느낌.
- 변경: `s = DASH_SPD * (0.45 + 0.55*t²)` — 초반 폭발, 후반 감속. 대시 종료 시 잔여 속도 50% 캐리.
- 결과: 프레임별 13.6 → 7.0 테이퍼 (측정치).

**대시 방향 즉시 확정**
- 대시는 turn 보간을 무시하고 현재 입력 방향으로 즉시 스냅. 회전 지연 때문에 원치 않는 방향으로 대시하는 문제 방지.

**테스트**
- Playwright 헤드리스 회귀 테스트로 4개 동작 모두 검증 (가속 램프 / 감속 활강 / 방향 보간 / 대시 테이퍼).
- JS 문법 검사 통과, 런타임 에러 없음.

### 튜닝 상수 (index.html)
```
ACCEL     = 0.28   // 높을수록 즉각적
DECEL     = 0.22   // 낮을수록 미끄러짐
TURN_RATE = 0.35   // 1.0 = 즉시 스냅
```
